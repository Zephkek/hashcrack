2 Attack Method

Choose how you want to attempt cracking the hash

Dictionary Attack

Recommended

Try passwords from a wordlist (dictionary). Fast and effective for common passwords.

⚡ Fast 📊 High Success Rate 💾 Low Resource Usage

Mask Attack

Use a pattern-based approach when you know the password structure (e.g., "word" + 2 digits).

🎯 Targeted ⏱️ Medium Speed 🧩 Pattern-Based

Brute Force

Slow

Try all possible combinations. Very thorough but can be extremely slow for longer passwords.

🔄 Exhaustive 🐌 Very Slow 🔋 High Resource Usage

Combination Attack

New

Combine words from two different wordlists (e.g., first name + last name, word + word).

🎯 Targeted ⚡ Fast 🔗 Combinatorial

Hybrid Attack

Advanced

Combine wordlist entries with mask patterns (e.g., dictionary word + 2 digits).

🧩 Flexible ⏱️ Medium Speed 🎨 Creative

Association Attack

Smart

Generate candidates based on context clues (username, email, filename, company name).

🧠 Intelligent ⚡ Fast 🎯 Context-Aware
